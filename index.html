⚠️ Browser Security Notice

This app needs to communicate with your local printer. If you're using HTTPS and encounter connection errors:

  1. Make sure your printer is on and connected to the same network
  2. For Chrome: Go to Site Settings and allow "Insecure content" for this site
  3. For Firefox: Click the shield icon in the address bar and disable protection for this site
  4. For Safari: This might not work due to stricter security policies
